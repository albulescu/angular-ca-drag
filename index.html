Drag & Drop Module for AngularJS

angular-ca-drag is a drag and drop module for AngularJS.

Features

  1. Basic drag and drop functionality
  2. Group drag items and drop target by keys
  3. Data to send over the drag events
  4. Custom callbacks for each drag event
  5. Cloned and custom drag indicator
  6. Drag feedback to specify if drop allowed or not

Api


Service
DragManager

Register drag elements, drop targets and handles the drop, feedback and other features

register(element) Register drag elements, drop targets and handles the drop, feedback and other features
addDropTarget(element, scope, over, complete) A decorator for dom element. Enable simple drag functionality
dragging Property that indicating the dragging is active
active Active dragging element
DraggableElement A decorator for dom element. Enable simple drag functionality

Directive
ca-drag-enabled Enable elment dragging
ca-drag-data Data from current scope to send it with events
ca-drag-type Specify the type of current dragging element. This is used when you drag multiple types in one page. Also you can specify on drop targets the ca-drop-type directive.
ca-drop-type Specify what kind of draggable elements this drop target will accept
ca-drag-clone The indicator to show when drag an element. By default an image is cloned from original element
ca-drop-accept Register this element as drop zone
ca-drag-begin(event) Event fired when dragging is started
ca-drag-move(event) Event fired when dragging
ca-drag-complete(event) Event fired when dragging is completed
ca-drop-complete(event) Add this to drop target and it will be fired when a draggable element is dropped
ca-drop-over(event) Add this to drop target and it will be fired when a draggable element is over the drop target
